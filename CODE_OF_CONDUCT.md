# Code of Conduct

This project follows the **Contributor Covenant, version 2.1**.

The full text of the Covenant is available at:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

By participating in this project — filing issues, opening pull requests,
or otherwise interacting with the community — you agree to uphold the
Covenant.

## Reporting

To report a Code of Conduct issue, contact the maintainers privately
via the security contact listed in [`SECURITY.md`](SECURITY.md).
Reports are handled confidentially.

## Scope

This Code of Conduct applies to all project spaces — the GitHub
repository, issues, pull requests, discussions, and any official
project communication channels — and to public spaces when an
individual is representing the project.
